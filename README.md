# FsSoftware

## Raspberry pi 3 + GT-521F52 Scanner

### Python3.7.2

#### - Installation 
````
  wget https://www.python.org/ftp/python/3.7.2/Python-3.7.2.tgz

  tar -xf Python-3.7.2.tgz
  cd Python-3.4.3/

  ./configure --enable-optimizations --with-ssl

  make && make altinstall
````
